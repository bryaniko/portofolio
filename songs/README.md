# Music Files Guide

## Cara menambahkan lagu ke musik player

Folder ini (`songs/`) adalah tempat menyimpan semua file audio untuk musik player.

### File yang diperlukan:

Tambahkan file MP3 dengan nama berikut ke folder ini:

1. **midnight-coding.mp3** - Lagu Lo-Fi pertama
2. **jazz-cafe.mp3** - Lagu Jazz
3. **ambient-dreams.mp3** - Lagu Ambient
4. **chillhop-vibes.mp3** - Lagu Chillhop
5. **lofi-study.mp3** - Lagu Lo-Fi kedua
6. **evening-jazz.mp3** - Lagu Jazz malam

### Cara Menambahkan File Audio:

1. **Siapkan file MP3 Anda** (atau format audio lainnya: WAV, OGG, M4A)
2. **Letakkan di folder ini** dengan nama sesuai daftar di atas
3. **Refresh halaman website** untuk mendengarkan lagu

### Format yang Didukung:

- MP3 (.mp3) - Paling kompatibel ✓
- WAV (.wav)
- OGG (.ogg)
- M4A (.m4a)

### Custom Upload (Alternatif):

Jika Anda ingin menambahkan lagu tambahan selain 6 lagu preset:
1. Klik tombol "➕ Add Music" di sidebar musik player
2. Pilih file audio dari komputer Anda
3. Lagu akan disimpan otomatis dan bisa diputar langsung

### Catatan:

- Pastikan file audio dalam format yang benar
- Ukuran file sebaiknya tidak terlalu besar (<10MB per file)
- Saat browsing folder ini, pastikan tidak ada subfolder, hanya file MP3
