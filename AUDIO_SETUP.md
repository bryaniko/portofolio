# 🎵 Setup Audio untuk Music Player

## ✅ Status Setup

Folder `songs/` sudah siap menerima file audio!

## 🎯 Yang Perlu Anda Lakukan:

### **Langkah 1: Siapkan File Audio**
Kumpulkan 6 file MP3 lagu pilihan Anda (atau bisa membeli dari platform musik)

### **Langkah 2: Rename File**
Rename file sesuai urutan di bawah:

| Nama File | Genre | Durasi Saran |
|-----------|-------|--------------|
| `midnight-coding.mp3` | Lo-Fi | 3-4 menit |
| `jazz-cafe.mp3` | Jazz | 3-4 menit |
| `ambient-dreams.mp3` | Ambient | 4-5 menit |
| `chillhop-vibes.mp3` | Chillhop | 3-4 menit |
| `lofi-study.mp3` | Lo-Fi | 3-4 menit |
| `evening-jazz.mp3` | Jazz | 4-5 menit |

### **Langkah 3: Letakkan di Folder**
Pindahkan file ke folder:
```
c:\Users\XREYZEN\Documents\malas\songs\
```

### **Langkah 4: Test**
1. Buka browser
2. Refresh halaman website
3. Klik tombol **▶ Play** di music player
4. Seharusnya sudah ada suara! 🎶

---

## 📝 Catatan Penting:

✓ Format audio yang didukung: **MP3, WAV, OGG, M4A**

✓ Anda bisa upload lagu tambahan pakai tombol **"➕ Add Music"** di sidebar

✓ Lagu custom akan tersimpan otomatis di browser

✓ Jika tidak ada suara, pastikan:
- File MP3 sudah ada di folder `songs/`
- Nama file **persis sama** seperti tabel di atas
- Browser sudah di-refresh
