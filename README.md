## 🎮 Tentang Website

Website ini adalah portfolio personal yang menampilkan karya-karya dan proyek-proyek saya di bidang pengembangan game dan seni 3D. Dibangun dengan teknologi web modern, website ini menampilkan:

### ✨ Fitur Utama

1. **Profil Pribadi**
   - Avatar dan identitas lengkap
   - Bio ringkas sebagai Game Developer & 3D Artist
   - Lokasi (Indonesia) dan kontak

2. **Introduksi Interaktif**
   - Penyampaian cerita tentang perjalanan karier
   - Highlight keahlian utama:
     - 🎮 **Game Development** - Unity, C#, Game Design
     - 🎨 **3D Art & Animation** - Blender, Substance Painter
     - 💻 **Creative Coding** - Shaders, Procedural Generation

3. **Music Player**
   - Built-in music player dengan visualizer
   - Playlist dengan berbagai genre musik
   - Fitur upload lagu sendiri
   - Visual vinyl record yang animatif

4. **Repository & Projects**
   - Pinned repositories dari GitHub
   - Daftar proyek lengkap
   - Statistik stars dan informasi lainnya

5. **Efek Visual Premium**
   - Starfall background animation
   - Gradient overlays yang dinamis
   - Smooth transitions dan hover effects
   - Responsive design untuk semua device

6. **Loading Animation**
   - Animasi loading yang menarik saat halaman dimuat
   - Progress bar dengan gradient animation
   - Transisi halus saat konten siap
